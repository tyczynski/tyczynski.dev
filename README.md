# Personal webpage - tyczynski.dev

## Getting Started

Run the development server:

```bash
# Run the development server
$ yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Scripts

```bash
# Run the development server
$ yarn dev

# Build the production version
$ yarn build

# Run builded production version
$ yarn start
```
